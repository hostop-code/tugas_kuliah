1. Running di php 8.1 bisa diunduh melalui link berikut: https://sourceforge.net/projects/xampp/files/XAMPP%20Windows/8.1.17/
2. Copy folder project ke xampp/htdocs atau public web server lainnya
3. Buat database baru pada PhpMyAdamin
4. Setting koneksi database pada file .env 
5. Import database.sql melalui PhpMyAdamin
8. Akses project anda pada browser misal : localhost/erp/public
9. Buat profile bisnis anda, username dan password dental klik register
10. Login dan aplikasi sudah bisa digunakan
